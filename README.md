# HVAC Cofounder Briefing

Open `index.html` in a browser. It is a standalone interactive decision brief built from the supplied `HVAC Contractors - Vertical Research Paper` dated June 11, 2026.

## Interactions

- Scenario switcher: planning, one-call-close, and upside.
- Economics lab: package presets plus closes, churn, ARPA, delivery cost, and rep cost.
- Five-state decision gate tracker persisted in local storage.
- Executive/full-detail view.
- Risk filters.
- Print / save as PDF.

## Research trail

The source synthesis, design decisions, content inventory, and QA reports live in `.research/`.
